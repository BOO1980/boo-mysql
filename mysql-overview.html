MySQL Overview

  • Databases are everywhere
  • By Defintion a database is a structured collection of data
  • The data relating to each other by nature, therefore we use the term relational database
  • In the relational database, we model data like products, categories, tags using tables
  • A table contains coloums rows, like a spreadsheet
  • A table may relate to another table using a relationship

SQL - the language of the relational database

  • SQL Stands for the structured query language
  • SQL is the standardized language used to access the database
  • ANSI/SQL defines the SQL standard.
  • SQL contains three parts
    • Data defintion language includes statements that help you define the database and its objects
    • Data manipulation language contains statments that allow you to update and query data
    • Data control language allows you to grant the permissons to a user to access specific data in the database

What is MySQL

  • MySQL is a database management systems that allows you to manage relational databases
  • It is open source software backed by oracle
  • You can use MySQL without paying
  • You can change MySQL source code to suit your needs
  • MySQL is an essential component of the LAMP stack, which includes Linux, Apache, MySQL and PHP
